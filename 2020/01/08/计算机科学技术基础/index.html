    计算机的分类

    根据计算机的效率，速度，价格，运行的经济性质和适应性划分，计算机可分为通用计算机和专用计算机两大类。

    1. 通用计算机功能齐全，通用性强，适应面广，可完成各种各样的工作，但是牺牲了效率，速度和经济性。
    2. 专用计算机是专门为某些特定问题而设计的功能单一的计算机，一般来说其结构要比通用计算机来得简单，具有可靠性高、速度快、成本低的优点，是最有效，最经济和最快速的计算机，但是其适应性很差。
    3. 通用计算机分类
      1. 超级计算机
      2. 大型机
      3. 服务器
      4. 工作站
      5. 微型机
      6. 单片机
    4. 第一台电子计算机 ENIAC 诞生于1946年在美国宾夕法尼亚大学。

    机器数与真值

    • 在计算机中，数值的正负号也是用二进制编码表示的

    • 通常把一个数的最高位定义为符号位，用 0 表示正，用 1 表示负，成为数符。其余位表示数值。

    • 把在计算机内存放的正、负号数码化的数称为机器数

    • 把机器外部由 “+” 、“-” 号表示的数称为真值。

    • 例如 （0000 1011）= （+11）

      ​ （1000 1011） = （-11）

    原码，反码与补码

    • 正数，原码，反码补码相同

    • 负数

      • 原码：符号位为1，数值位为绝对值的二进制数

        例 [-109]=11101101

      • 反码：将原码初符号位外，逐位取反。

        例 [-109]=10010010

      • 补码：将反码末位加 1

        例[-109]=10010011

      • 补码求原码：除符号位外，逐位取反，末位加一。

    计算机硬件系统

    计算机硬件系统由运算器，控制器，存储器，输入设备和输出设备5部分组成

    组成：

    主机：中央处理器cpu，内存储器，总线，输入/输出接口

    外部设备：外存储器（硬盘，光盘，u盘等），输入设备（键盘，鼠标，扫描仪，数码相机等），输出设备（显示器，打印机，绘图仪等），其他（网卡，声卡，调制解调器，视频卡等）

    计算机工作原理：输入设备在控制器的帮助下，将数据和程序传入给内存储器，运算器在控制器的帮助下，从内存储器中读取和使用数据，计算完毕后将数据传输给输出设备。

    计算机软件系统组成

    系统软件：操作系统，数据库管理系统，语言处理系统，服务程序（诊断程序，调试程序，编辑程序等）
    应用软件：为解决各类实际问题而设计的程序系统称为应用软件。从其服务对象的角度,又可分为通用软件和专用软件两大类。

    ​ 通用软件：通常是为解决某一类问题而设计的，而这类问题是很多人都要遇到和解决的。如：文字处理，表格处理电子演示等

    ​ 专用软件：为解决某一问题而设计的软件。

    程序设计语言的基本概念

    1. 基本概念：
      • 低级语言：又称面向机器语言，特点计算机系统的固定语言
      • 汇编语言：是机器语言的一种提升，它使用了一些助记符来表示机器指令中的操作码和操作数。
      • 高级语言：与人的自然语言十分接近，使用起来很方便，极大的提高了程序设计效率。
      • 编译程序：也称编译器，将源程序翻译成目标语言程序，然后在计算机上运行目标程序。虽然执行效率高，但是编写出来的程序可读性差，且难以修改和维护。
      • 解释程序：又称解释器，它或者直接执行源程序，或者将源程序翻译成某种中间代码后执行。

    面向对象基本概念：面向对象技术是一种已对象为基础，以事件或消息来驱动对象执行的处理程序设计技术。它以数据为中心而不是以功能为中心，数据相对功能而言具有更强的稳定性。

    面向对象技术强调在软件开发过程中面向客观世界或问题域的事物，采用人类在认识客观世界过程中普遍采用的思维方法，直观、自然地描述客观世界中的有关事物。

    面向对象三大特性：封装继承多态。

    大数据技术基础知识

    1. 什么是大数据

      ​ 一种规模大到在获取，存储，管理，分析方面大大超出了传统数据库软件工具能力范围的数据集合，具有海量的数据规模，快速的数据流转，多样的数据类型和数据价值密度低四大特征。大数据需要特殊的技术，以有效处理大量的容忍经过时间内的数据。适用于大数据的技术包括大规模并行处理（MPP）数据库，数据挖掘，分布式文件系统，分布式数据库，云计算平台，互联网和可扩展的存储系统。

      最小的基本单位是bit，按顺序给出所有单位：bit，byte，KB，MB，GB，TB，PB，EB，ZB，YB，BB，NB，DB。

    2. 大数据的特征

      1. 容量（volume）:数据的大小决定所考虑的数据价值和潜在的信息。
      2. 种类（variety）：数据类型的多样性。
      3. 速度（velocity）：指获得数据的速度。
      4. 可变性（variability）：妨碍了处理和有效管理数据的过程。
      5. 真实性（veracity）：数据的质量。
      6. 复杂性（complexity）：数据量巨大，来源渠道多。
      7. 价值（value）：合理运用大数据，以低成本创造高价值。

    3. 大数据技术包括哪些

      1. 数据收集：大数据采集主要有四种来源：管理信息系统，web信息系统，物理信息系统，科学实验系统。
      2. 数据存取：大规模的结构化数据，半结构化和非结构化数据，结构化和非结构化混合的大数据。
      3. 基础架构：云存储，分布式文件存储等。
      4. 数据处理：将来自不同数据集的数据收集，整理，清洗，转换后，生成到一个新的数据集，为后续查询和分析处理提供统一的数据视图。
      5. 统计分析：假设检验，显著性检验，差异分析，相关分析，T检验，方差分析 …

    物联网技术基础知识

    1. 物联网概念

      物联网（Internet of Things）指的是将无处不在(Ubiquitous)的末端设备（Devices）和设施（Facilities），包括具备“内在智能”的传感器、移动终端，工业系统，数控系统，家庭智能设施，视频监控系统等、和“外在使能”(Enabled)的，如贴上RFID的各种资产（Assets）、携带无线终端的个人与车辆等等“智能化物件或动物”或“智能尘埃”（Mote），通过各种无线和/或有线的长距离和/或短距离通讯网络实现互联互通（M2M)、应用大集成（Grand Integration)、以及基于云计算的SaaS营运等模式，在内网（Intranet）、专网（Extranet）、和/或互联网（Internet）环境下，采用适当的信息安全保障机制，提供安全可控乃至个性化的实时在线监测、定位追溯、报警联动、调度指挥、预案管理、远程控制、安全防范、远程维保、在线升级、统计报表、决策支持、领导桌面（集中展示的Cockpit Dashboard)等管理和服务功能，实现对“万物”的“高效、节能、安全、环保”的“管、控、营”一体化。

    2. 物联网技术的四大支撑技术

      1. 射频识别（RFID）
      2. 无线传感网络（WSN/ZigBee）
      3. 传感器技术
      4. 智能服务
      5. M2M
      6. 两化融合，工业信息化，自控控制化

    3. 物联网的两大技术范畴

      1. 智能网络空间技术
      2. 物联网终端技术

    4. 体系架构

      自下而上分别是感知层、网络层、应用层

      感知层实现物联网全面感知的核心能力，是物联网中关键技术、标准化、产业化方面亟需突破的部分，关键在于具备更精确、更全面的感知能力，并解决低功耗、小型化和低成本问题。网络层主要以广泛覆盖的移动通信网络作为基础设施，是物联网中标准化程度最高、产业化能力最强、最成熟的部分，关键在于为物联网应用特征进行优化改造，形成系统感知的网络。应用层提供丰富的应用，将物联网技术与行业信息化需求相结合，实现广泛智能化的应用解决方案，关键在于行业融合、信息资源的开发利用、低成本高质量的解决方案、信息安全的保障及有效商业模式的开发。

    云计算技术基础

    1. 云计算概念

      云计算是一种按使用量付费的模式，这种模式提供可用的、便捷的、按需的网络访问，进入可配置的计算机资源共享池（资源包括网络，服务器，存储，应用软件，服务），这些资源能够被快速提供，只需要提供很少的管理工作，或与服务商进行很少的交互。

      7. 极其廉价
      8. 潜在的危险性

    3. 云计算的服务

      1. laaS：基础设施即服务
      2. PaaS：平台即服务
      3. SaaS：软件即服务

    计算机病毒

    1. 定义：指编制者在计算机程序中插入破坏计算机功能或者破坏数据，影响计算机使用并且能够自我复制的一组计算机指令或者程序代码。

    2. 特性

      1. 繁殖性
      2. 破坏性
      3. 传染性
      4. 潜伏性
      5. 隐蔽性
      6. 可触发性

    3. 结构

      1. 引导模块（主控模块）：

        实现将计算机病毒引入计算机内存，并使得传染和表现模块处于活动状态。引导模块需要提供自我保护功能，避免在内存中 的自身代码不被覆盖或清除。计算机病毒程序引入内存后为传染模块和表现模块设置相应启动条件，以便在适当的条件下激活传染模块或触发表现模块。

      2. 感染模块

      3. 破坏模块（表现模块）：

      4. 触发模块

    4. 计算机病毒的分类

      1. 破坏能力分类：无害型，无危险型，危险型，非常危险型。
      2. 存在媒体分类：网络病毒，文件病毒，引导型病毒。
      3. 病毒算法分类：伴随型，蠕虫型，寄生型，练习型，诡秘型，变型。

    Internet的连接与服务功能

    1. internet的连接

      ICS是（internet连接共享）是windows系统内置的一种网络连接共享服务，它可以使家庭网络或小型办公室网络用户非常容易的连接到internet。

    2. 要使用ICS，有几点需要注意：

      (1)启用ICS的计算机必须具有两个网络接口：一个连接到内部局域网，通常是网卡;一个连接到Internet，通常是Modem或ISDN接口。

      (2)要配置ICS，必须具有Administrators组权限。

      (3)ICS设置完成后，本地的网络将使用动态的地址分配机制，因此不应该将此功能与其他Windows 系统Server DC、DNS服务器、网关、DHCP服务器，或配置为静态IP地址的系统一起使用。

      (4)在启用ICS后，连接到家庭或小型办公室网络的适配器将获得新的IP地址配置。有两种情况，一种是系统建议的使用动态分配的方式，这时启用ICS的计算机就充当了一台DHCP服务器;另一种是手动设置的方式。

    3. Internet的服务功能

      1. WWW浏览
      2. FTP与Telnet服务
      3. 电子邮件
      4. IP电话
      5. 网上寻呼机
